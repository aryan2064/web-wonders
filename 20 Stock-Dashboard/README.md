# Stock Dashboard


## 🌟 Overview

A stock tracking dashboard powered by the Alpha Vantage API. Search stocks by symbol or company name, view a market overview with top gainers, losers, and most active stocks, and manage a personal watchlist.

## ✨ Features

*   Search by stock symbol or company name
*   Market overview with top gainers, losers, and most active
*   Watchlist tracking
*   Stock price and change display
*   API key authentication
