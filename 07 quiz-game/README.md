# Quiz Game

![Project Banner](Quiz-Game%20(1).png)

## 🌟 Overview

A multiple-choice quiz game with progress tracking, score display, and a result screen with feedback. Answer questions, track your progress, and see how you did at the end.

## ✨ Features

*   Multiple choice questions
*   Progress bar showing question advancement
*   Score tracking throughout the quiz
*   Result screen with performance feedback
*   Restart capability

## 🚀 Live Demo / Deployment

Open `index.html` in your browser.
