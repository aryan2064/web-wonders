# QR Code Generator


## 🌟 Overview

Generate QR codes from any text or URL using a public QR API. Download the generated QR code image for offline use.

## ✨ Features

*   Generate QR codes from text or URLs
*   Download QR codes as images
*   Enter key support for quick generation
