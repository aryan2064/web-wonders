# Todo App


## 🌟 Overview

A simple and intuitive task manager. Add tasks, mark them as complete, and delete them when you're done. Supports the Enter key for quick task entry.

## ✨ Features

*   Add new tasks
*   Mark tasks as complete
*   Delete tasks
*   Enter key support for quick entry

## 🚀 Live Demo / Deployment

Open `index.html` in your browser.

## ▶️ Usage

1. Open `index.html` in any modern browser.
2. Type a task and press Enter or click add.
3. Click on a task to mark it complete, or click delete to remove it.
