# Typing Game


## 🌟 Overview

A typing game where words move across the screen. Type them correctly to shoot and score points. Words get faster as you progress — don't let them reach the end!

## ✨ Features

*   Progressive difficulty (words speed up over time)
*   Score tracking
*   Lives system
*   Word spawning with lane management
