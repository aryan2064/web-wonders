# Daily Quote Generator


## 🌟 Overview

Fetches random quotes from an API with a local fallback. Click to get a new quote and copy it to your clipboard with a smooth fade animation.

## ✨ Features

*   Fetch random quotes from the DummyJSON Quotes API
*   Fallback to local quotes if the API is unavailable
*   Copy quotes to clipboard
*   Fade animation on quote change
