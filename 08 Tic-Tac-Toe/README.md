# Tic-Tac-Toe

![Project Banner](Tic-Tac-Toe%201.png)

## 🌟 Overview

A classic two-player tic-tac-toe game with win and draw detection. The winning line is highlighted, and players can restart the game at any time.

## ✨ Features

*   Two-player gameplay (X and O)
*   Win detection with line highlighting
*   Draw detection
*   Restart game button

## 🚀 Live Demo / Deployment

Open `index.html` in your browser.
