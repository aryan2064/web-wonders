# Sudoku Solver


## 🌟 Overview

A fully interactive Sudoku game with difficulty selection, automatic puzzle generation, and an auto-solve feature powered by a backtracking algorithm.

## ✨ Features

*   Three difficulty levels (easy, medium, hard)
*   Automatic puzzle generation
*   Auto-solve using backtracking algorithm
*   Interactive grid for manual input
