# GitHub Finder


## 🌟 Overview

Search for GitHub users by username and view their profile including avatar, bio, stats, and latest repositories. Uses the GitHub REST API for real-time data.

## ✨ Features

*   Search GitHub users by username
*   Display profile with avatar, bio, and stats
*   Show latest repositories with language, stars, and forks
*   Company, blog, and Twitter info display
