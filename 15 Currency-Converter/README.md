# Currency Converter

![Project Banner](Currency%20Converter.png)

## 🌟 Overview

Convert between 150+ world currencies using real-time exchange rates from the ExchangeRate API. Simply select your currencies and amount to get instant conversions.

## ✨ Features

*   Convert between 150+ currencies
*   Real-time exchange rates
*   Dropdown currency selection

## 🚀 Live Demo / Deployment

Open `index.html` in your browser.

## ▶️ Usage

1. Open `index.html` in any modern browser.
2. Select the source and target currencies from the dropdowns.
3. Enter an amount to see the converted value.
