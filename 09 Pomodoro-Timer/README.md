# Pomodoro Timer

## 🌟 Overview

A Pomodoro timer with configurable work and break durations. Automatically switches between work and break modes, with an audio alert when the mode changes.

## ✨ Features

*   Configurable work and break durations
*   Auto-switch between work and break modes
*   Audio alert on mode switch
*   Visual mode indicator (work vs. break)

## 🚀 Live Demo / Deployment

Open `index.html` in your browser.

## 🛠️ Technologies Used

*   HTML5
*   CSS3
*   JavaScript
*   Web Audio API

## ▶️ Usage

1. Open `index.html` in any modern browser.
2. Set your desired work and break durations.
3. Start the timer. It will auto-switch between modes with an audio alert.

## 🧠 Learning Outcomes & Challenges

*   Timer lifecycle management with mode switching
*   Web Audio API for generating alert sounds programmatically
*   Syncing UI state with timer mode transitions
