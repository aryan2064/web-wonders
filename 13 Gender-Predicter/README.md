# Gender Predictor


## 🌟 Overview

Predicts gender based on a given name using the Genderize API. Displays the predicted gender, confidence percentage, and the number of data points behind the prediction.

## ✨ Features

*   Name-based gender prediction
*   Confidence percentage display
*   Data point count showing sample size
*   Async/await API calls
