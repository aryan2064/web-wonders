# Pokemon Viewer


## 🌟 Overview

Search for any Pokemon by name or ID using the PokeAPI. Displays official artwork, types, and base stats. Includes a random Pokemon button and quick suggestion buttons for easy exploration.

## ✨ Features

*   Search by name or ID
*   Random Pokemon button
*   Quick suggestion buttons
*   Official artwork display
*   Type and stat display
