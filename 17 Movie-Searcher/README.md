# Movie Searcher


## 🌟 Overview

Search for movies using the OMDb API. Results are displayed in a grid with posters. Click any movie to see detailed information including rating, genre, and director in a modal.

## ✨ Features

*   Search movies by title
*   Pagination with load more
*   Detailed movie info modal
*   Poster display
*   Rating, genre, and director info
