# Mandelbrot Set

![Project Banner](mandelbrot%20set%20ss%20(1).png)

## 🌟 Overview

An interactive Mandelbrot set fractal viewer built with the Canvas API. Click anywhere on the fractal to zoom in, use the zoom out button to step back, and reset to return to the default view. Uses a smooth coloring algorithm with HSL to RGB color mapping.

## ✨ Features

*   Click to zoom into the fractal
*   Zoom out button
*   Reset view
*   Smooth coloring algorithm
*   HSL to RGB color mapping
